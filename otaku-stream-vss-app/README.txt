To Start Production Application 

1 - go to the directory otaku-stream-vss-app
2 - npm i install --> (install all necessary packages)
3 - npm run clean --> (clean the dist build if it exists)
4 - npm run build --> (build the dist)
5 - npm run start --> (starts the server using the dist build for front end)
